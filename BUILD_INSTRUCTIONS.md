# Инструкция по компиляции, компоновке, установке и запуску

## 1. Требования

Для сборки и запуска программы должны быть установлены:

- Docker;
- Docker Compose.

Проверка установки:

```bash
docker --version
docker compose version
```

## 2. Переход в каталог сборки

Открыть терминал в каталоге, где находится данный файл.

В текущей директории должен находиться файл:

```text
docker-compose.yml
```

## 3. Компиляция, компоновка и установка

Выполнить команду:

```bash
docker compose up --build
```

При выполнении команды:

- компилируется серверная часть;
- собирается клиентская часть;
- создаются контейнеры, сеть и том для базы данных;
- запускаются сервер, клиент и PostgreSQL.

Миграции базы данных выполняются автоматически при запуске серверного контейнера.

## 4. Запуск

После успешного выполнения команды открыть в браузере:

```text
http://localhost:5173
```

Если страница открылась, программа запущена.

## 5. Остановка

Для остановки нажать в терминале:

```text
Ctrl+C
```
