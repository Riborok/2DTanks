# 2DTanks Frontend - Онлайн клиент

Это фронтенд-клиент для онлайн-версии игры 2D Tanks.

## Установка

```bash
npm install
```

## Запуск

Для запуска в режиме разработки:

```bash
npm run serve
```

Приложение будет доступно на `http://localhost:3000`

## Сборка

Для сборки в режиме разработки:

```bash
npm run build-dev
```

Для сборки в режиме продакшена:

```bash
npm run build-prod
```

## Подключение к серверу

По умолчанию клиент подключается к `ws://localhost:8080`. 

Чтобы изменить адрес сервера, отредактируйте файл `src/ts/online/WebSocketClient.ts` и измените значение по умолчанию в конструкторе:

```typescript
constructor(url: string = 'ws://your-server:8080')
```

## Структура

- `src/ts/components/ui/` - React компоненты для UI (экран подключения, выбор танка, лобби, игра)
- `src/ts/online/` - Логика для онлайн-режима (WebSocket клиент, типы, рендерер)
- `src/ts/sprite/` - Система рендеринга спрайтов
- `src/ts/geometry/` - Геометрические утилиты
- `src/img/` - Изображения для игры

## Особенности

- **Тонкий клиент**: Вся игровая логика выполняется на сервере, клиент только отображает состояние
- **Рендеринг**: Используется существующая система рендеринга из оригинальной игры
- **WebSocket**: Реальное время обновление состояния игры через WebSocket
